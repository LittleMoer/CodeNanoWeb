# Branding Portfolio Template

A modern branding portfolio built with Vite, React, TypeScript, and Tailwind CSS.

## Tech Stack
- React + React Router
- TypeScript
- Vite
- Tailwind CSS

## Quick Start (Windows PowerShell)

```powershell
# From project root
npm install
npm run dev
```

Open the URL shown (e.g., http://localhost:5173).

## Build & Preview
```powershell
npm run build
npm run preview
```

## Customize
- Update colors and fonts in `tailwind.config.js`.
- Replace placeholder images and copy in components under `src/components` and pages in `src/pages`.
- Add real form handling for `Contact` using your preferred backend/service.

## Project Structure
- `src/components`: Header, Footer, Hero, PortfolioGrid, Testimonial, CTA
- `src/pages`: Home, About, Work, Services, Contact
- `index.html`: SEO meta + fonts
- `tailwind.config.js`: brand tokens
